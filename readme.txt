Customer Coupons for WooCommerce helps you display your coupons on the website.

== Description ==

Customer Coupons for WooCommerce is a WooCommerce extension which helps you introduce coupons to customers. The plugin will displays available coupons on single product pages, cart page and checkout page. Inform customers that there are coupons available to use. And let customers know how much they can save if use coupons. Urge customer to use coupons and purchase products.  The plugins also make it easier to use coupons by display a “Use Coupon” button. Customers just need to click on coupons which they want to use.

###FEATURES

&#9658; **COUPON LABEL**: The coupon label informs which coupons are available to use.

 -**Coupon label**: display coupon label on product, cart, check out page.

 -**Enable/Disable**: select if a coupon will be displayed.

 -**Title**: enter the title of the coupon label. By default, the title will be the coupon value.

 -**Show on**: schedule a time to show a coupon to customers.

 -**Term**: display term of use on coupon labels.

 -**Preview**: preview the coupon label design in your backend.

 -**APPLY COUPON button**: display an apply coupon button on coupon labels. Make it is easier to use coupons.

&#9658; **MANAGE COUPON**: WooCommerce Customer Coupons helps you to manage your easily. With the coupon list and filter coupon by category option.

 -**Coupon list**: manage coupon in backend with a coupon list. The coupon list provides code, type, amount, description,

 -**Sort coupon by allowed categories

&#9658; **TEMPLATES**

 -**4 front-end templates to choose from.

 -**Background color.

 -**Background gradient color.

 -**Title color.

 -**Expired date color.

 -**Term of use color.

 -**APPLY COUPON button text color.

 -**APPLY COUPON button background color.

 -**Position: select a position to display the coupon label on single product pages. There are 6 positions for you to choose from.

&#9658; **SEND EMAIL**: When the admin adds new or edit a coupon and allow specific email addresses to use that coupon. WooCommerce Customer Coupons will send emails to those email address to inform that they are allowed to use a coupon.

- **Position:**select coupon position on single product page.

- **Template:** there are 4 coupon templates. Each template have different color settings.

&#9658; **Email:**Sending coupon email to customers. Email includes coupon code, valid time.

== Installation ==

1. Unzip the download package
1. Upload `woo-customer-coupons` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
